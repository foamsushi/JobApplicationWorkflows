### **Rule Set 9: The "Bold List" Prohibition**

The single most common visual indicator of ChatGPT/Claude output is the **Vertical List with Bold Inline Headers**.1

* **The Artifact:**  
  * **Efficiency:** The system improves speed.  
  * **Scalability:** The system grows with you.  
  * **Security:** The system protects data.  
* **The Wikipedia Ruling:** This is flagged as "Inline-header vertical lists" and "Overuse of boldface".1 It breaks the narrative flow and looks like a PowerPoint slide, not a report.  
* **The Human Rule:** **Use Paragraphs.** Weave the points into a continuous narrative. If you must use a list, use simple bullets without the bold headers, or format it as a proper data table if it contains structured data.
